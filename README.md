🤝 Sistema de Gestão de ONG — POO em Java
Projeto acadêmico desenvolvido como desafio prático de Programação Orientada a Objetos, simulando o núcleo (Core) de um sistema real de gestão para uma ONG. Aplica os três pilares fundamentais da POO: Encapsulamento, Herança e Composição.

🎯 Objetivo
Atuar como Engenheiro de Software construindo um sistema que modela os membros, doadores e projetos sociais de uma ONG, demonstrando na prática como os pilares da POO resolvem problemas reais de organização e reutilização de código.
